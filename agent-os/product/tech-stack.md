# Tech Stack

## Frontend

- React with TypeScript
- Vite (build tool)

## Backend

- Java 25
- Spring Boot 4
- Gradle (multi-project Kotlin DSL monorepo)

## Database

- PostgreSQL (primary datastore) — Docker for local dev
- Redis (caching / sessions) — Docker for local dev
- Flyway (database migrations)

## Infrastructure

- Local dev: Docker (PostgreSQL + Redis)
- Production: AWS

## Integrations

- Google Ads API (current)
- Microsoft Ads API (planned)
- Meta (Facebook) Ads API (planned)
- Yahoo Ads Japan API (planned)
